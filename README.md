## Deskripsi
Praktikum ini membahas penggunaan fitur multimedia
pada aplikasi Android meliputi audio, video, kamera,
dan pengelolaan file multimedia.

---

## Fitur Multimedia
- Audio Recorder
- Audio Player
- Video Player
- Kamera & Galeri
- File Multimedia Management

---

## Tools & Teknologi
- Kotlin
- Jetpack Compose
- MediaRecorder
- MediaPlayer
- ExoPlayer
- GitHub
